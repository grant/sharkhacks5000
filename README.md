# sharkhacks5000
The home page for SharkHacks5000 https://sharkhacks5000.com 

## Development
### Start
```sh
yarn
npm start
```

### Develop
Tab 1 (javascript):
```sh
npm watch
```
Tab 2 (server):
```sh
gulp
```

### Deploy
```sh
git push heroku master
heroku open
```
