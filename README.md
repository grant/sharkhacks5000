# sharkhacks5000
The home page for SharkHacks5000 https://sharkhacks5000.com 

## Development
### Start
```sh
yarn
npm start
```

### Deploy
Using the [heroku toolbelt](https://devcenter.heroku.com/articles/heroku-cli).
```sh
# heroku git:remote -a <app-name>
git push heroku master
heroku open
```

## Open
```sh
npm run open
```
